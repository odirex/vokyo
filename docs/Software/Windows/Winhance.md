# Winhance

**Link:** [https://winhance.net](https://winhance.net)

**Description:**
An open-source utility designed to debloat, optimize, and customize Windows 10 and 11 systems. It allows users to manage applications via WinGet, remove unwanted features, and adjust privacy settings.
