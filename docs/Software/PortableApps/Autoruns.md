# Autoruns

**Link:** [https://learn.microsoft.com/sysinternals/downloads/autoruns](https://learn.microsoft.com/sysinternals/downloads/autoruns)

**Description:**
A Microsoft Sysinternals utility that shows exactly what programs are configured to run automatically during system boot or login.

**WinGet Command:**
`winget install Microsoft.Sysinternals.Autoruns`
