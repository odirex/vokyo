# BleachBit

**Link:** [https://www.bleachbit.org/](https://www.bleachbit.org/)

**Description:**
A free, open-source disk space cleaner and privacy manager that deletes junk files, cookies, and temporary data.

**WinGet Command:**
`winget install BleachBit.BleachBit`
