# Microsoft Store Apps

**Link:** [https://apps.microsoft.com/store/detail/systemtraymenu/9N24F8ZBJMT1?hl=en-us&gl=us](https://apps.microsoft.com/store/detail/systemtraymenu/9N24F8ZBJMT1?hl=en-us&gl=us)

**Description:**
SystemTrayMenu&#xD;&#xA;Browse and open your files easily&#xD;&#xA;&#xD;&#xA;SystemTrayMenu is an open source &#x27;Desktop Toolbar&#x27; or &#x27;Start Menu&#x27; alternative. It offers a clear, personalized menu which can be controlled via keyboard, mouse or touchscreen. Files, links and folders are organized in several levels as dropdown menus. Various options like appearance, hotkey, autostart and behavior can be adjusted.&#xD;&#xA;&#xD;&#xA;Shortcuts to folders or to network paths are resolved. Their content is displayed like a folder.&#xD;&#xA;The search function allows you to define hotkey sequences to find and open items.&#xD;&#xA;The menu stays open, if starting several applications and mouse still on the menu or if a script steals the focus for a short time.&#xD;&#xA;Menu appears transparent in case it has no focus and closes automatically when you leave the menu.&#xD;&#xA;A scrollbar appears when there are too many items, which can also be controlled via mouse wheel.&#xD;&#xA;Supports drag and drop to open an item with an application or to copy it to the desktop.&#xD;&#xA;There is a possiblity to create URL shortcuts including the browser icon via drag and drop the URL from the browser into the menu.
