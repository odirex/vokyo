# PuTTY

**Link:** [https://www.putty.org/](https://www.putty.org/)

**Description:**
A free, open-source terminal emulator and network file transfer application supporting SSH, Telnet, and other protocols.

**WinGet Command:**
`winget install PuTTY.PuTTY`
