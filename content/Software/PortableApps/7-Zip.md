# 7-Zip

**Link:** [https://www.7-zip.org/](https://www.7-zip.org/)

**Description:**
A free, open-source file archiver with high compression ratios, supporting various formats like 7z, ZIP, and TAR.

**WinGet Command:**
`winget install 7zip.7zip`
