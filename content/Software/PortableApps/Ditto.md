# Ditto

**Link:** [https://ditto-cp.sourceforge.io/](https://ditto-cp.sourceforge.io/)

**Description:**
An advanced, open-source clipboard manager that allows you to store and recall multiple copied items.

**WinGet Command:**
`winget install ScottBax.Ditto`
